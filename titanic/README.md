\# Titanic



\## Objetivo

Explorar o dataset do Titanic e entender quais fatores influenciam a sobrevivência dos passageiros.



\## Dataset

Dataset clássico do Titanic, amplamente utilizado para aprendizado em ciência de dados.



\## Etapas

\- Análise exploratória dos dados

\- Limpeza e tratamento de dados

\- Modelo simples de classificação
